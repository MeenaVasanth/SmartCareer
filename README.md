# SmartCareer
Learners and job-seekers face an overwhelming number of course and certification options. This AI-powered recommender helps you choose the right next step by personalizing learning paths based on your background, technical readiness, and career aspirations.

#Key Features
Personalized Recommendations: Get course suggestions tailored to your profile

Skill Gap Analysis: Understand what skills you need to develop

Learning Timeline: Short-term and long-term planning

Justified Recommendations: Clear explanations for each suggestion

Multiple Input Methods: Manual input, resume upload, or LinkedIn profile
